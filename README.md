# Fishlog
A frontend for the FishLog app. Built with Javascript, Express JS, Sass and Ion Rangeslider.

## Install dependencies
`npm install`

## Run linter
`npx eslint src/js`

## Build CSS
`npm run css`

## Build the app
`npm run build`

## Run in development mode
`npm run dev`

## Run in production mode
`npm start`
